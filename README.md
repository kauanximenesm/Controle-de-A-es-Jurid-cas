CJuris - Controle de Ações Jurídicas

Este projeto é um sistema de gestão jurídica focado no cadastro e acompanhamento de processos legais. Desenvolvido como parte da disciplina de **Desenvolvimento de Aplicações Corporativas**.

O objetivo é oferecer uma interface operacional clara para advogados e escritórios gerenciarem prazos, status de ações e informações de processos em andamento.

🚀 Sprint 1: Hospedagem e Landing Page
Nesta primeira fase, o foco foi a estruturação da identidade visual, organização de pastas e deploy contínuo.
- [x] Estrutura de pastas organizada (`/css`, `/js`, `/assets`).
- [x] Landing page responsiva.

- [x] Deploy realizado no Vercel.

🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica.
* **CSS3**: Estilização moderna com Flexbox.
* **JavaScript**: Lógica de interação do dashboard.
